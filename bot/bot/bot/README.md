# Trading Bot – Binance Futures Testnet

A simplified Python trading bot that places MARKET, LIMIT, and STOP_LIMIT orders on the Binance Futures Testnet (USDT‑M). Built for the Application Task.

## Features
- Place **MARKET**, **LIMIT** and **STOP_LIMIT** orders.
- Support **BUY** and **SELL** sides.
- CLI interface with validation.
- Structured code with separate API client and order logic.
- Comprehensive logging (console + file).
- Error handling for API, network, and validation errors.
